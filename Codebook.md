# Summary
This code book provides brief description of fields used in `tidy.txt` file representing the result of the Coursera's Data Cleaning course project.

# Data source
* https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

# Fields Description

## Primary identifiers of observations

* `subject` - Primary identifier of observed subject
* `activity` - A type of activity when subject was wearing a smartphone (Samsung Galaxy S II)
